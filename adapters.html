Alfa AWUS036ACH — Dual‑band, good range

Alfa AWUS036ACHLinux • Windows

What is it for? A dual‑band USB Wi‑Fi adapter (2.4 & 5 GHz) used to add strong wireless capability. Useful for packet capture or monitor mode in labs when drivers support it.

Simple setup

  • Windows: Plug in → install Alfa/Realtek drivers if needed.
  • Linux / Kali: Plug in → install `rtl8812au` (or DKMS patched driver) → check `ip a` / `iwconfig`.
  • VM: USB passthrough → install drivers in guest.

Quick checks: ip a, iwconfig, airmon-ng.

Legal: Use only on networks you own or have permission to test.

Alfa AWUS036NHA — Stable monitor mode & injection

Alfa AWUS036NHALinux • Windows

What is it for? A USB Wi‑Fi adapter known for reliable monitor mode and packet injection support — great for wireless auditing in controlled labs.

Simple setup

  • Windows: Plug in → install drivers if needed.
  • Linux / Kali: Kernel usually provides ath9k_htc → verify with ip a / iwconfig → use airmon-ng.
  • VM: USB passthrough + drivers in guest.

Quick checks: ip a, iwconfig, try monitor mode in a safe lab.

Legal: Only for authorized testing and learning.

TP‑Link TL‑WN722N — Cheap & useful (v1 only)

TP‑Link TL‑WN722NLinux • Windows

What is it for? A budget USB Wi‑Fi adapter. Note: only version v1 (Atheros AR9271) reliably supports monitor mode and injection. v2/v3 are standard adapters.

Simple setup

  • Check the version on the label (v1 = AR9271).
  • Windows: Plug in → install TP‑Link drivers if needed.
  • Linux / Kali (v1): ath9k_htc driver should work → verify with lsusb / iwconfig.
  • VM: USB passthrough to guest + drivers there.

Quick checks: lsusb (look for AR9271), iwconfig, airmon-ng.

Legal: Use only in your lab or with explicit permission.
