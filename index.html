Kali Linux

  • Kali Linux — a Debian-based penetration-testing distro packed with hundreds of security tools (Nmap, Metasploit, Wireshark, Aircrack-ng, etc.).
  • Legal note: For learning & lab use only. Use on VMs or networks you own/have permission to test.
  • If you want to download click here!
  • Keep your system up to date — run sudo apt update && sudo apt full-upgrade -y regularly
  • Install tool groups with Kali metapackages (e.g., kali-linux-default, kali-linux-wireless) to get focused tool sets
  • Learn with labs ( TryHackMe /Hack The Box), Kali docs, and community guides — practice in isolated environments only.

Burp Suite

  • Burp Suite — web-app security toolkit for intercepting, inspecting, and modifying HTTP(S) traffic.
  • Community (free) for manual testing; Professional/Enterprise (paid) for automated scans and advanced features.
  • Key tools: Proxy (intercept), Repeater (resend/tweak), Intruder (fuzz/bruteforce), Scanner (automated vuln scans).
  • Typical setup: run Burp, point browser proxy to 127.0.0.1:8080 (use FoxyProxy to switch easily).
  • Install on Kali: use package manager or download from PortSwigger for latest version.
  • Great for learning web pentesting — but only test sites you own or have permission to test.
  • Pro tip: save interesting requests to Repeater or Sequencer for deeper manual analysis.

Nmap

  • Nmap — a powerful network scanning tool used to discover hosts, open ports, and services on a network.
  • Can detect OS types, versions, and running services; great for mapping and auditing networks.
  • Supports multiple scan types: TCP SYN scan (-sS), UDP scan (-sU), ping scan (-sn), and more.
  • You can use Nmap on Kali Linux or any Linux distro , Windows/macOS too.
  • Always scan networks you own or have permission to test — unauthorized scanning is illegal.
  • Can combine with other tools like Zenmap (GUI) or scripts (--script) for vulnerability enumeration.

Metasploit

  • Metasploit — a powerful exploitation framework used by security pros to develop, test, and run exploits against vulnerable systems.
  • Contains thousands of modules: exploits, payloads, auxiliary (scanners/fuzzers), and post‑exploitation tools.
  • Meterpreter is a popular payload that gives an interactive post‑exploit shell for advanced testing and modules
  • Comes as the open-source Metasploit Framework and a paid Metasploit Pro with extra automation and reporting.
  • Often used in labs and pentesting to validate vulnerabilities — useful for creating reproducible tests and demos.
  • Included in Kali Linux by default (or installable via package manager); also available from Rapid7.
  • Always use Metasploit only on systems you own or have explicit permission to test — unauthorized exploitation is illegal.

Wireshark

  • Wireshark — a network protocol analyzer that captures and displays packets in real time for deep traffic inspection.
  • Great for troubleshooting, learning protocols (TCP/IP, HTTP, DNS), and analyzing captures (.pcap/.pcapng).
  • Use capture filters (set before capture) to limit recordings and display filters (after capture) to focus on specific traffic (e.g., http, ip.addr==1.2.3.4).
  • Works on Linux/Windows/macOS; on some OSes you need elevated privileges to capture live traffic.
  • Save captures to .pcap and share with teammates or import into other tools for analysis.
  • Powerful features: packet reassembly, protocol decoding, follow TCP stream, statistics, and custom dissectors.
  • Always capture only on networks you own or have permission to analyze — intercepting other people’s traffic can be illegal and invade privacy.
  • Pro tip: use capture filters to reduce noise (e.g., port 80 or port 443) and learn display filters — they speed up analysis.

Hashcat

  • Hashcat — a fast, GPU-accelerated password-recovery tool used to crack password hashes for recovery and security testing
  • Supports many hash types (MD5, NTLM, bcrypt, etc.) and multiple attack modes (dictionary, combinator, mask, rule-based).
  • Best used with a good GPU — dramatically faster than CPU-based cracking.
  • Common workflow: export hashes from your own test systems, then run Hashcat in a controlled lab to check password strength.
  • Example (conceptual): hashcat -m -a hashes.txt wordlist.txt — replace placeholders and only run on hashes you own.
  • Pro tip: use rule files and masks to simulate realistic password patterns (helps assess real-world password strength).
  • Always ethical: only test passwords you own or have permission to audit.

Hydra

  • Hydra — fast network login tester for services like SSH, FTP, HTTP, and more.
  • Used to validate lockouts and rate-limiting in controlled environments (lab-only).
  • Example (lab-only): hydra -L users.txt -P passwords.txt ssh://127.0.0.1.
  • Always test only on systems you own or have explicit permission to audit.
